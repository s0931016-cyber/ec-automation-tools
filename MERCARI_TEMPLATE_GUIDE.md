# Mercari Listing Template Guide

このファイルは、iPodカスタム商品のメルカリ用説明文を生成するための標準テンプレートです。

Codexはメルカリ説明文を短く簡略化しすぎず、以下の構成を基本にすること。

---

## 対象世代

このテンプレートは第7世代固定ではなく、以下の世代で共通利用する。

- iPod classic 第7世代
- iPod classic 第6世代 / 6.5世代
- iPod Video 第5世代 / 5.5世代

実装では `generation` や `model` の入力値から自然な表記を生成する。

例。

```text
iPod 第7世代をベースに256GB SSDへ換装したカスタムモデルです。
iPod 第6世代をベースに128GB SSDへ換装したカスタムモデルです。
iPod 第5.5世代をベースに256GB SSDへ換装したカスタムモデルです。
```

モデルナンバーは必須ではない。入力がある場合だけ補足として使う。

---

## 標準タイトル

タイトルは世代、容量、主なカスタム内容から生成する。

### 例

```text
【大容量256GB】iPod classic 第7世代 SSD換装 USB-C
【大容量128GB】iPod classic 第6世代 SSD換装 USB-C
【大容量256GB】iPod 第5.5世代 SSD換装 USB-C
```

第7世代256GB USB-Cの価格目安。

- 通常: 36,500円
- 早めに売る場合: 35,800円

## 標準説明文構成

メルカリ説明文は、最低限この構成を保つ。

1. 挨拶
2. ベース機種とカスタム内容
3. HDDからSSDへ変更した説明
4. Type-Cポート説明
5. 状態 / 動作確認
6. 新品交換部品
7. その他
8. 発送・保証
9. 必要に応じてハッシュタグ

## モデルナンバーの扱い

モデルナンバーは必須入力にしない。

### 入力がない場合

```text
iPod 第7世代をベースに256GB SSDへ換装したカスタムモデルです。
```

### 入力がある場合

必要に応じて以下のように補足してもよい。

```text
iPod 第7世代（MC293J）をベースに256GB SSDへ換装したカスタムモデルです。
```

ただし、モデルナンバーが不明な場合に `{modelNumber}` のようなプレースホルダーを文面に残してはいけない。

---

## 文面生成ルール

### 簡略化しすぎない

以下のような短い説明だけで終わらせない。

```text
256GB SSD、USB-C、2000mAhバッテリー搭載のiPodです。
```

これは不可。

---

## 仕様ごとの差し替え

### reversibleUsbC が true の場合

```text
両面認識にも対応しているので
純正dockコネクタ（片面）の煩わしさから解放されます。
```

### reversibleUsbC が false の場合

```text
データ転送は片面のみの認識となります。ご理解お願いいたします。
```

### dataTransfer が true の場合

```text
・USB-Cデータ転送
```

### Taptic Engine が true の場合

【その他】またはSSD説明後に以下を追加する。

```text
Taptic Engine搭載しております。元のクリック音は撤去し、コツコツっとした振動のみ伝わる仕様です。
```

### iMod が true の場合

5th / 5.5th系の場合のみ、音質カスタムとして追記する。

## 実装時の注意

- メルカリ文面は日本語で自然にする
- 第7世代固定にしない
- 5世代 / 5.5世代 / 6世代 / 7世代に対応する
- モデルナンバーは任意扱いにする
- `{modelNumber}` などの未置換プレースホルダーを出力しない
- 過度に広告っぽくしない
- 事実と違う仕様を入れない
- 商品状態が不明な場合は断定しすぎない
- 画像で確認してほしい傷や状態がある場合は「詳細は画像をご確認ください」を入れる
- eBay英語説明文とは別テンプレートとして扱う
